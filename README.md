# NoteTakerExtensionChrome


Noted is a Chrome extension that aims to simplify the process by allowing the user to highlight portions of web pages and save them to their account as they surf the web. Moreover, users can take their own notes on the highlighted portions! Notes will be kept nice and tidy in the Noted web app for the next time they're needed. 

### Technologies used:  
* React 
* jQuery 
* Node.js 
* Express 
* Webpack 
* MongoDB 
* Mongoose 
* HTML 
* CSS
* Heroku
